A todo application for Rails 5.0 beta on Ruby 2.3-preview1

## Database creation

```
$ bin/rails db:create
```

## Database initialization

```
$ bin/rails db:migrate
```

## Usage

```
$ bin/rails s
```

## How to run the test suite

```
$ bin/rspec
```
