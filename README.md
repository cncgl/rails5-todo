A todo application for Rails 5.0 beta on Ruby 2.3.0

## Database creation

```
$ bin/rails db:create
```

## Database initialization

```
$ bin/rails db:migrate
```

## Usage

```
$ bundle exec rackup -p 3000 config.ru
```
visit http://localhost:3000

## How to run the test suite

```
$ bin/rake
```
